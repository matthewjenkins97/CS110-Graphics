Detailed Description

An object representing an action from the user.

Used by EventHandler objects. User actions that create Event objects include:

  • Pressing/Releasing a key on the keyboard
  • Pressing/Releasing a button on the mouse while on a GraphicalObject with an event handler
  • Moving the mouse while on a GraphicalObject with an event handler

Each of these actions will call their corresponding methods in EventHandler automatically and give an instance of Event to the method called.
