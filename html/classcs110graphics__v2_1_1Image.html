window- Window - the window which the object will be added to
image_loc- str- The file location for an image, see below for instructions regarding file locations
width- int - (default: 100) the width of the image
height- int - (default: 100) the height of the image
center- tuple of (int * int) - (default: (200, 200)) the center location for the image

File locations:

  • If a file is in the same folder/directory as the program, just use the name of the file
  • Otherwise use a file path: eg. "~/110/bots/images/bot.jpg" or "./images/bot.jpg"

Note that "." represents the current directory and ".." represents the parent directory.
